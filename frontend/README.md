# Frontend (React + Vite)

## Setup
```
npm install
```

## Configure API base URL
Create a `.env` file in this folder based on `.env.example`:
```
VITE_API_BASE_URL=http://localhost:8000
```

## Run dev server
```
npm run dev
```
Open http://localhost:5173

## Build for production
```
npm run build
npm run preview
```
